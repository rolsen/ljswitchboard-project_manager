We have finally begun to release updated versions of Kiplingv3.1 and there are several great improvements. We are also about to release a new version of our website which makes this release different from other Kipling releases as the build & distribution process is changing. A quick list of all of the changes is below:

Improvements

  • Improved device selector
  • Improved module navigation
  • MUCH better error handling & reporting
  • Improved register matrix that now remembers the registers you have selected.
  • Improved lua script eding that remembers what script you last had open.
  • Improved start-up time & Splash screen
  • New project repository

Things that have yet to be done

  • Kipling update manager

Things that will probably break when we release a new website

  • Device updater-from-web
